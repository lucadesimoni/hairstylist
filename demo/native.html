💇 Native API booking

The widget in mode="native-inline" — booking, availability and registration run through hairlist-api.js. On a real site you pass data-api-key; this page injects a small mock transport so it runs offline.

Embed code (real site)

<div id="booking"></div>
<script src="hairlist-api.js"></script>
<script src="hairlist-widget.js"
  data-shop="steiner"
  data-mode="native-inline"
  data-mount="#booking"
  data-api-key="YOUR_SALON_API_KEY"
  data-color="#b8860b"></script>
